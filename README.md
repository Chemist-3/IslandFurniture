## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

This are the things you need to built and run the project. 

* Glassfish Server
* mySQL (run the SQL script found in the project here)

If you do not know how to setup, follow this video guide [here](https://drive.google.com/drive/folders/0Bw551lbxyagYdkdqUFQwYkN6UXc).
You should get prerequisites functional before building the project.

### Installation and Build

1. Clone the repo
```sh
git clone https://github.com/Snowdrifft/IslandFurniture.git
```
2. Open both IS3102_Project-Student and IS3102_WebServices-Student projects in NetBeans IDE.
3. Clean and Build IS3102_Project-ejb if necessary.
4. Run IS3102_Project-Student, wait till browser popup before procceding to next step.
5. Deploy IS3102_WebServices-Student.
